# ILM-ORA - AI-Powered Degree & University Recommendation Platform

A modern, student-friendly platform for degree and university recommendations with AI-powered matching, gamified RIASEC career quiz, and comprehensive career insights.

## 🎨 Features

- **Landing Page** - Modern hero section with features showcase
- **Authentication** - Login/Signup with validation
- **Dashboard** - Personalized student dashboard with quick actions
- **RIASEC Quiz** - Gamified career personality assessment
- **Degree Recommendations** - AI-powered degree matching (90%+ accuracy)
- **University Recommendations** - Comprehensive university search with filters
- **Career Insights** - Salary data, growth trends, and market demand
- **Sentiment Dashboard** - Student feedback analysis with charts
- **Scholarship Finder** - Pakistan & international scholarships
- **Hostel Finder** - University accommodation search
- **Feedback System** - Rating and review submission
- **Admin Panel** - Content management dashboard
- **🌐 Bilingual Support** - Full English & Urdu (اردو) with RTL layout

## 🚀 Tech Stack

- **React 18** with TypeScript
- **Vite** for blazing fast builds
- **Tailwind CSS v4** for styling
- **React Router** for navigation
- **Motion (Framer Motion)** for animations
- **Recharts** for data visualization
- **shadcn/ui** components
- **Lucide React** icons

## 📦 Installation

1. **Install Dependencies:**
```bash
npm install
```

2. **Run Development Server:**
```bash
npm run dev
```

3. **Build for Production:**
```bash
npm run build
```

4. **Preview Production Build:**
```bash
npm run preview
```

## 🌐 Language System

ILM-ORA supports **bilingual functionality**:

- **English (Default)**: Left-to-right layout
- **Urdu (اردو)**: Right-to-left layout with Noto Nastaliq font
- **Language Toggle**: Similar to theme toggle (globe icon in nav)
- **Persistent**: Language choice saved in localStorage
- **267 Translation Keys**: Covering all major UI elements

See `LANGUAGE_GUIDE.md` for implementation details and `TRANSLATION_STATUS.md` for current progress.
